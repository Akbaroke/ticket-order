## About App

Ticket-id is an online platform application that provides bus ticket booking services.
<br/>
"Enjoy Your Seamless Bus Ticketing Experience!"

<br/>

## Tech Stack

- Reactjs Vite
- TailwindCss
- CI4
- MySql


<br/>

## Features

- Find Schedule
- Select Schedule
- Select seat
- Buy a ticket
- Payment
